Initially tells how web browsers interact to the html code.

<style>
Can create and define CSS code inside
an html file. Such as:

    color: adds and configure which color is in the text.
            can also use hex code of color,
    font-size: resize the size of the text
    margin: adds spacing between edges
    background-color: adds and configure which color for background
    text-align: aligns the text in the display

.title {

}
